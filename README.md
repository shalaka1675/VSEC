# VSEC Practicals Repository

This repository contains simple, easy-to-understand code examples for VSEC practicals.

## Files Included

### 1. hello.py
A simple greeting program that demonstrates basic function definition and usage.

**Features:**
- Simple function with string parameter
- String formatting using f-strings
- Main execution block

### 2. calculator.py
Basic calculator with arithmetic operations (add, subtract, multiply, divide).

**Features:**
- Multiple simple functions
- Error handling for division by zero
- Function documentation
- Example usage in main block

### 3. string_operations.py
String manipulation utilities for common operations.

**Features:**
- String reversal
- Vowel counting
- Palindrome checking
- Word capitalization
- Demonstrates list comprehension and string methods

## How to Use

Run any file directly:
```bash
python hello.py
python calculator.py
python string_operations.py
```

Or import functions in your own code:
```python
from calculator import add, multiply
result = add(5, 10)
```

## Learning Concepts

These examples demonstrate:
- Function definition and calling
- Parameters and return values
- String manipulation
- Conditional statements
- Error handling
- List comprehension
- Documentation strings
